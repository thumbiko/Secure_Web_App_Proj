# 🚗 X-Hausted Autos – Secure Web Application

## 📌 Project Overview
X-Hausted Autos is a secure web application developed to manage automotive service bookings.  
The platform allows users to register, log in, and book services such as starlight installation, ambient lighting, diagnostics, and valet services.

The application is designed with security in mind and follows best practices for modern web development.

---

## ✨ Features
- User registration and authentication  
- Role-based access control (Admin & User)  
- Booking system (Create, Read, Update, Delete)  
- Service management  
- Secure API structure  

---

## 🛠️ Technologies Used
- **Backend:** Node.js (Express)  
- **Database:** MongoDB (Mongoose)  
- **Version Control:** Git & GitHub  

---

## 🔐 Security Features
- Password hashing using bcrypt  
- JSON Web Token (JWT) authentication  
- Role-based authorization  
- Input validation and sanitization  
- Secure error handling  

---

## 🏗️ System Architecture
The application follows a modular architecture:

- **Models** → Define database structure  
- **Routes** → Handle API endpoints  
- **Controllers** → Contain business logic  
- **Middleware** → Manage authentication and security
